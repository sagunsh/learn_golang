# Learning Go (Golang)

Repository for learning Golang for someone coming from Python/PHP background.

### Running your code

    go run basics.go

### Compile and execute

    go build hello_world.go
    ./hello_world.go

### Initialize module (to install 3rd party libraries)

    go mod init <module name>
